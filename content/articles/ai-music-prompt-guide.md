# AI音乐提示词完全指南：如何写出高质量的Prompt

提示词（Prompt）是AI音乐生成的关键。好的提示词能让AI准确理解你的意图，生成符合预期的作品。本文将系统讲解提示词写作技巧。

## 提示词基础结构

### 标准格式

一个完整的AI音乐提示词通常包含：

```
[歌曲结构]
歌词内容

[风格描述]
音乐风格、BPM、人声类型等参数
```

### 示例

```
[Verse 1]
Walking down the empty street
Memories beneath my feet

[Chorus]
We were young and wild and free
Now it's just a memory

[Style: Indie Pop, Female Vocal, 110 BPM, Melancholic]
```

## 歌曲结构标签

### 常用结构标签

**基础结构**：
- `[Intro]` - 前奏
- `[Verse]` - 主歌
- `[Pre-Chorus]` - 副歌前
- `[Chorus]` - 副歌
- `[Bridge]` - 桥段
- `[Outro]` - 尾奏

**进阶结构**：
- `[Hook]` - 记忆点
- `[Drop]` - 电音降落
- `[Break]` - 间奏
- `[Interlude]` - 过渡段
- `[Solo]` - 独奏段

### 结构组合示例

**流行歌曲标准结构**：
```
[Intro]
[Verse 1]
[Pre-Chorus]
[Chorus]
[Verse 2]
[Pre-Chorus]
[Chorus]
[Bridge]
[Chorus]
[Outro]
```

**电子舞曲结构**：
```
[Intro]
[Build-up]
[Drop]
[Break]
[Build-up]
[Drop]
[Outro]
```

## 提示词写作技巧

### 1. 具体而非笼统

❌ **不好的例子**：
```
[Style: Pop music]
```

✅ **好的例子**：
```
[Style: Upbeat Indie Pop, Female Vocal, 120 BPM, 
Acoustic Guitar, Synth Pads, Energetic, Summer Vibe]
```

### 2. 使用多个描述词

**单一描述**：
```
[Style: Sad]
```

**多维度描述**：
```
[Style: Melancholic Indie Folk, Soft Male Vocal, 
Acoustic, Rainy Day, Introspective, 85 BPM]
```

### 3. 风格组合创新

**经典组合**：
- Pop + Electronic = Synth Pop
- Rock + Hip Hop = Rap Rock
- Jazz + Electronic = Nu Jazz
- Classical + Electronic = Neoclassical

**创意组合**：
```
[Style: Cyberpunk Synthwave meets Traditional Chinese, 
Epic, Cinematic, 140 BPM]
```

### 4. 参考具体艺人

可以参考知名艺人的风格：

```
[Style: In the style of Billie Eilish, 
Dark Pop, Whispered Vocals, Minimalist Production]
```

```
[Style: Taylor Swift inspired, 
Country Pop, Storytelling, Catchy Hooks]
```

### 5. 场景化描述

描述音乐适用的场景：

```
[Style: Perfect for a sunset beach party, 
Tropical House, Relaxed, 115 BPM, Summer Vibes]
```

```
[Style: Late night driving music, 
Synthwave, Nostalgic, Neon Lights, 100 BPM]
```

## 常见错误和解决方案

### 错误1：提示词过于简单

❌ **问题**：
```
[Style: Pop]
Love song
```

✅ **改进**：
```
[Verse 1]
Your smile lights up my darkest days
In your eyes I find my way

[Chorus]
You're my sunshine, my guiding star
No matter where, no matter how far

[Style: Romantic Pop Ballad, Male Vocal, Piano-driven,
80 BPM, Emotional, Strings, Heartfelt]
```

### 错误2：风格冲突

❌ **问题**：
```
[Style: Heavy Metal, Calm, Relaxing, Soft]
```

✅ **改进**：
```
[Style: Heavy Metal, Aggressive, Powerful, 
Distorted Guitars, 160 BPM, Intense]
```

### 错误3：结构混乱

❌ **问题**：
```
Some lyrics here
More lyrics
[Chorus] maybe?
Random words
```

✅ **改进**：
```
[Intro]
[Verse 1]
First verse lyrics
[Chorus]
Chorus lyrics
[Verse 2]
Second verse lyrics
[Chorus]
[Outro]
```

### 错误4：歌词与风格不匹配

❌ **问题**：
```
[Style: Death Metal]
Flowers blooming in the spring
Butterflies with gentle wings
```

✅ **改进**：
```
[Style: Death Metal]
Chaos reigns in darkest night
Warriors prepare to fight
```

## 不同工具的提示词差异

### Suno AI
- 支持最复杂的提示词
- 可以使用 `[Style: ...]` 标签
- 歌词和风格分开描述

### Udio
- 更自由的格式
- 可以上传参考音频
- 支持更细致的参数调整

### MusicGen
- 更简单的描述
- 主要依赖文本描述
- 不支持复杂结构标签

1. **清晰的结构**：使用标准标签
